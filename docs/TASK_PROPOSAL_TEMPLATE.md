## â“ Common Questions

**Q: Can I assign points myself?**
A: No, only admins can assign point labels. This ensures consistency across all tasks.

**Q: How long until admin reviews my task?**
A: Usually within 24 hours. Urgent tasks can be messaged directly to admin.

**Q: Can I edit a task after creating it?**
A: Yes! Click on the task in Vikunja and edit any field except point labels.

**Q: What if I complete a task early?**
A: Mark it as complete in Vikunja as soon as you finish. Points are awarded on completion.

**Q: Can I see my total points?**
A: Point tracking system is coming soon. For now, tasks show individual point values.

**Q: What if task takes longer than expected?**
A: Update the due date in Vikunja or message admin if points should be adjusted.
